# DNA to Protein Analysis Pipeline

This project is a bioinformatics pipeline that reads DNA sequences from FASTA files and performs biological analysis.

## Features

- Reads DNA sequences from FASTA file
- Calculates sequence length
- Computes GC Content
- Computes AT Content
- Counts nucleotide bases (A, T, G, C)
- Converts DNA → RNA
- Converts RNA → Protein
- Finds the longest protein sequence
- Saves results into output file

## Project Structure

DNA_Protein_Project/

data/ → FASTA files  
src/ → Python source code  
results/ → Output results  
README.md → Project description  

## Requirements
